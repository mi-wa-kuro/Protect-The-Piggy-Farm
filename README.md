# Protect The Piggy Farm! 🐷

ぶたさんたちがのびのびと暮らす平和な農場に、いたずら好きのもぐらたちがやってきました。

プレイヤーはハンマーを使ってもぐらを退治し、ぶたさんたちの平和を守ります。

GitHub Pagesで公開中です。

## ゲームURL

https://mi-wa-kuro.github.io/Protect-The-Piggy-Farm/

## ゲーム紹介

Protect The Piggy Farm! は HTML・CSS・JavaScript だけで制作したオリジナルのもぐらたたきゲームです。

単純にもぐらを叩くだけでなく、ぶたさんを守りながらプレイすることでゲーム性を高めています。

## 遊び方

* もぐらを叩く → +1点
* 子豚を叩く → -1点
* ヘルメットもぐらは1回では倒せない
* 制限時間内にできるだけ高得点を目指そう

## 実装した機能

* タイトル画面
* スターウォーズ風ストーリー演出
* カウントダウン演出
* 効果音・BGM
* スコア表示
* ゲームオーバー画面
* 子豚アニメーション
* レスポンシブ対応（スマホ対応）
* GitHub Pagesによる公開
