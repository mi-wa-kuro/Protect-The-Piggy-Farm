# Protect The Piggy Farm! 🐷

ぶたさんたちがのびのびと暮らす平和な農場に、いたずら好きのもぐらたちがやってきました。

プレイヤーはハンマーを使ってもぐらを退治し、ぶたさんたちの平和を守ります。

GitHub Pagesで公開中です。

## ゲームURL

https://mi-wa-kuro.github.io/Protect-The-Piggy-Farm/

## ゲーム紹介

Protect The Piggy Farm! は HTML・CSS・JavaScriptだけで制作したオリジナルのもぐらたたきゲームです。

単純にもぐらを叩くだけではなく、子豚やヘルメットもぐら、金色のレアもぐらなど複数のキャラクターを登場させ、ゲーム性を高めています。

## 遊び方

* もぐらを叩く → +1点
* 子豚を叩く → -1点
* ヘルメットもぐらは2回叩くと倒せます
* 金色のレアもぐらを叩く → +5点
* 制限時間内にできるだけ高得点を目指そう
